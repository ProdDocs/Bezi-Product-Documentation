## Using the Text tool

1. To activate the Text tool press `t` + `spacebar`

2. To draw a text box, click and drag anywhere in your scene.

3. Type in the text box. Now it's a Text object.

## Customizing Text objects

<AccordionGroup>
  <Accordion title="Editing contents of a Text object" defaultOpen={false}>
    Double click on the Text object and type the new content.
  </Accordion>

  <Accordion title="Adjusting Text object properties" defaultOpen={false}>
    The following Text object properties can be adjusted in the right sidebar.

    1. Typeface

    2. Weight or Style

    3. Text Size

    4. Letter spacing

    5. Line height

    6. Horizontal alignment within the text box

    7. Vertical alignment within the text box

    8. Letter case (Original case, Upper case, Lower case, Sentence case)
  </Accordion>

  <Accordion title="Uploading custom fonts" defaultOpen={false}>
    Click the **import font** icon in the right sidebar. The accepted custom font file formats are .otf, .ttf, and .woff.
  </Accordion>
</AccordionGroup>
