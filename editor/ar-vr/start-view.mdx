## Using start view

<Steps>
  <Step title="Step 1">
    Make sure there is at least one Camera object in the scene. See the [Cameras documentation](https://docs.bezi.com/editor/cameras/cameras) for directions on how to add a Camera object to a scene.

    ![](/images/start-view-camera.png)
  </Step>

  <Step title="Step 2">
    Select the **Share** icon in the top toolbar.
  </Step>

  <Step title="Step 3">
    Open the **Prototype Settings** section of the Share popover.

    ![](/images/start-view-settings.png)
  </Step>

  <Step title="Step 4">
    Select the dropdown next to 'Start View'.

    ![](/images/select-start-view-camera.png)
  </Step>

  <Step title="Step 5">
    Select the Camera object to set as the starting point.


  </Step>
</Steps>
