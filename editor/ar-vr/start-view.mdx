## Using start view

<Steps>
  <Step title="Step 1">
    Make sure there is at least one Camera object in the scene. See the [Cameras documentation](https://docs.bezi.com/editor/cameras/cameras) for directions on how to add a Camera object to a scene.
  </Step>

  <Step title="Step 2">
    Select the **Share** icon in the top toolbar
  </Step>

  <Step title="Step 3">
    Select **Play Settings** at the top of the popover
  </Step>

  <Step title="Step 4">
    Select the dropdown next to 'Start View'
  </Step>

  <Step title="Step 5">
    Select the Camera object to set as the starting point&#x20;
  </Step>
</Steps>
