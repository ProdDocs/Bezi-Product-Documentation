## Using start view

1. Make sure there is at least one Camera object in the scene

   1. See Cameras documentation \[page link] for directions on how to add a Camera object to a scene

2. Select the **Share** icon in the top toolbar

3. Select **Play Settings** at the top of the popover

4. Select the dropdown next to 'Start View'

5. Select tje Camera object to set as the starting point&#x20;
