## Adding materials to the Material Library

1. Select the object

2. Click on the **Library** icon in the right sidebar (shown beside the material type)

3. In the Material Library popover, click on the **+** icon

4. Name your material

5. Press `enter` to save the material to the Library

6. The material will now be visibile in the Material Library popover

  <Accordion title="Replacing Library materials" defaultOpen={false}>
    To replace an existing material with another one from the Material Library:

    1. Select the material you want to replace

    2. Click the **Replace Material** icon in the right sidebar (next to the material name)

    3. Select the material you want to use from the Material Library popover
  </Accordion>
