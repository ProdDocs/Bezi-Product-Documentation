title: "Export Editor Scenes"
description: "Move scenes outside of the Editor. Exported files will not retain any interactive pieces, just the 3D scene and assets."
---

## Supported formats

Editor scenes can be exported in GLTF, USDZ, or STL format.

## Exporting instructions

1. Select the **Bezi** logo in the top toolbar

2. Hover over **File** in the dropdown

3. Select **Export file** from the File dropdown

4. Select the desired export file format

5. Click **Export** to start the file download in the browser
