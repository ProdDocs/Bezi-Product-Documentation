Streamline collaboration and standardize components used within a team or a project. Create a shared asset library that only members of a team can save to and pull from.&#x20;

## Using the Team Library

<AccordionGroup>
  <Accordion title="Accessing the Team Library" defaultOpen={false}>
    1. Open the desired Editor file

    2. Select **Library** from the bottom bar to open the Library panel

    3. Select the dropdown next to 'File Library'&#x20;

    4. Select **Team Library** from the dropdown

    5. Team assets will be visibile under the **Components** section. They can be dragged and dropped into a scene from there
  </Accordion>

  <Accordion title="Adding components to the Team Library" defaultOpen={false}>
    The Editor supports FBX, GLB, or GLTF formats. Assets over 100MB may impact scene performance.

    1. Open the a Team file following the above steps

    2. Import/create an asset, or select an asset you have in your scene

    3. Right click and select 'Create component'

    4. Components created within Team files will appear in the Team Library
  </Accordion>

  <Accordion title="Managing assets in the Team Library" defaultOpen={false}>
    Components stored in the Team Library can be edited or deleted in the original file that the component was created in. You can navigate to the main component by right clicking on the instance in the scene and selecting 'Go to main component'.
  </Accordion>

  <Accordion title="Team Library permissions" defaultOpen={false}>
    Components uploaded to a Team Library are available for any members of the team to view and add into a scene.
  </Accordion>
</AccordionGroup>
