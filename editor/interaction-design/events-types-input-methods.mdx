## Using events

Events are actions that trigger animations. Add interactions to objects using the State Machine \[prod doc link] and define events that trigger animations using the Timeline \[prod doc link].&#x20;

## Event types

<AccordionGroup>
  <Accordion title="Auto" defaultOpen={false}>
    Animation is triggered automatically when the scene is loaded.
  </Accordion>

  <Accordion title="On pointer down" defaultOpen={false}>
    Animation is triggered when the pointer is pressed.&#x20;

    How the 'On pointer down' event works with different input methods:

    * Mouse: Left click

    * Touch: Tap

    * VR controllers: Index finger triggers (Quest)

    * Hands: Index finger and thumb pinch
  </Accordion>

  <Accordion title="On pointer up" defaultOpen={false}>
    Animation is triggered when the pointer is released.&#x20;

    How the 'On pointer up' event works with different input methods:

    * Mouse: Left click

    * Touch: Tap

    * VR controllers: Index finger triggers (Quest)

    * Hands: Index finger and thumb pinch
  </Accordion>

  <Accordion title="On hover inside" defaultOpen={false}>
    Animation is triggered when the pointer hovers on the object.
  </Accordion>

  <Accordion title="On hover outside" defaultOpen={false}>
    &#x20;Animation is triggered when the pointer exits the object.
  </Accordion>

  <Accordion title="On key down" defaultOpen={false}>
    Animation is triggered when the viewer presses the assigned key. This event requires the viewer to interact with a keyboard.
  </Accordion>

  <Accordion title="On key up" defaultOpen={false}>
    Animation is triggered when the viewer releases the assigned key. This event requires the viewer to interact with a keyboard.
  </Accordion>

  <Accordion title="Direct touch (Body Rig)" defaultOpen={false}>
    Direct touch is a Body Rig property that places a collider on the fingertips of a viewer's tracked hands.&#x20;

    To access 'Direct touch' add a Body Rig to the scene. It will be enabled by default.&#x20;
  </Accordion>
</AccordionGroup>
