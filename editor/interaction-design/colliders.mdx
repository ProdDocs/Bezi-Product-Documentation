## Using colliders

<AccordionGroup>
  <Accordion title="Turning on colliders" defaultOpen={false}>
    1. Select any object, empty, or light in the Editor scene&#x20;

    2. Go to the Properties Panel&#x20;

    3. Find the "Colliders" section

    4. Press the **+** button to add a collider to the scene&#x20;
  </Accordion>

  <Accordion title="Viewing colliders in a scene" defaultOpen={false}>
    When colliders are turned, on a rainbow box outline will represent them in the scene. You can have multiple colliders in a scene or on a single object.
  </Accordion>
</AccordionGroup>

## Collider types

In Bezi their are two types of colliders; solid colliders and trigger colliders. By default, colliders  default to the trigger collider. Click the **dotted circle** icon to switch collider types.&#x20;
