Use Bezi's **in-depth project** (scripts, assets, scenes, packages, etc.) and Unity knowledge to find, interpret, and fix bugs faster than manual or code-only investigation.

## How can I use Bezi to find and fix bugs? 

1. Define what you want the tool to do
2. Prompt Bezi with the right details (Steps 1-5, **below**) 
3. Bezi will suggest what's causing the bug and how to fix it

## What types of bugs can Bezi help with? 

1. Console errors
2. Behavioral issues 
3. Optimization
