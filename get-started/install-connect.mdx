## Install and set-up Bezi

<Steps>
  <Step title="Install the Bezi app">
    1. Visit [bezi.com](http://bezi.com) \> select `Download` \> choose your operating system
    2. Run the installer
    3. Launch the Bezi app

    Add the app to your dock for ease of access.
  </Step>
  <Step title="Set-up your account">
    1. Open the Bezi app
    2. Select `Log In / Sign Up` and use email or Google to sign-in
    3. When set-up is complete, you'll see a webpage that says "Launch Bezi App"
    4. Click `Launch app` to return to Bezi
    5. Start your free trial (50 credits, no expiration)
  </Step>
</Steps>

## Connect Bezi with Unity

<Steps>
  <Step title="Open the Bezi app">
    Confirm your app is open and you see the Get Started screen.
  </Step>
  <Step title="Open a Unity project ">
    1. Open the Unity project you want to connect
    2. Confirm the Unity project is in a folder with write permissions
    3. Return to Bezi
    4. Open the **Project Picker** dropdown in Bezi's menu bar \> select your project
       1. If you don't see the desired project in the dropdown, click `Add Project` to manually add the Unity project to Bezi
       2. If you can't find your Unity project, see [this documentation](https://docs.bezi.com/bezi/troubleshooting#why-am-i-not-seeing-my-project-in-the-projects-dropdown%3F)
    1. Check name in the **Project Picker** to confirm the right Unity project is connected

    <Tip>
      Both the Unity project and Bezi app must be open to use them together. We recommend a split screen to see both simultaneously.
    </Tip>
  </Step>
  <Step title="Install Bezi's Unity plugin" stepNumber={3}>
    Select the plugin `Install` button, located in step 2 of the Get Started flow.

    <Tip>
      If you run into trouble installing the plugin via the Bezi app, download the Unity plugin directly from our [Changelog](https://docs.bezi.com/bezi/product/changelog).
    </Tip>
  </Step>
  <Step title="Grant Unity permission to install" stepNumber={4}>
    After you've clicked `Install` in Bezi, return to the Unity project. Unity will ask for consent to install the plugin. Once granted, the Unity Package Manager popup will appear.

    You may also see a _"_**_Consent Request_**_"_ popup from Unity. It's completely safe to accept, as it only modifies some APIs in Bezi to match your Unity version. No personal files are affected.
  </Step>
  <Step title="Confirm connection status">
    To confirm the connection is established, check Unity's menu for a green dot next to the **Bezi icon** and Bezi's menu for green dot next to the **Unity icon**.

    If either is red, follow [these steps to fix a broken connection](https://docs.bezi.com/bezi/troubleshooting#how-do-i-reinstate-a-lost-connection-between-unity-and-bezi%3F).

    <Warning>
      Both connection indicators **must** be green for Bezi to work.
    </Warning>
  </Step>
  <Step title="Sync your project with Bezi" stepNumber={6}>
    First time you connect Bezi to a new Unity project, you must wait for it to index the entire project. Do not prompt Bezi until this step is completed. If you prompt before it's indexed, you risk interrupting the sync and receiving bad responses.
