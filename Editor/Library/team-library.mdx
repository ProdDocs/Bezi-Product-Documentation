Streamline collaboration and standardize assets used within a team or a project. Create a shared asset library that only members of a team can save to and pull from.&#x20;

## Using the Team Library \[cecilia confirm this is correct & how to use assets from team folder?]

<AccordionGroup>
  <Accordion title="Accessing the Team Library" defaultOpen={false}>
    1. Go to the Editor Home, also known as ‘Files page’ (where all files, teams, and account settings can be accessed)

    2. Navigate to the desired Team in the left side bar

    3. Select the **Assets** folder under the team&#x20;

    4. This folder contains all assets shared among the Team
  </Accordion>

  <Accordion title="Adding assets to the Team Library" defaultOpen={false}>
    The Editor supports FBX, GLB, or GLTF formats. Assets over 100MB may impact scene performance.&#x20;

    1. Open the desired Team Asset folder following the above steps

    2. Select the **Add assets** button in the top right corner

    3. Find the asset

    4. Asset will only be visible in the Team's Asset folder, they will not show up in the Team's Files folder
  </Accordion>

  <Accordion title="Managing assets in the Team Library" defaultOpen={false}>
    Assets stored in the Team Library can be linked to, duplicated, renamed, and deleted. Right click on the asset to open a dropdown and select the desired action.&#x20;
  </Accordion>

  <Accordion title="Updating assets in the Team Library (is this possible??)" defaultOpen={false} />

  <Accordion title="Team Library permissions" defaultOpen={false}>
    Assets uploaded to a Team Library are available for any members of the team to view and add into a scene.
  </Accordion>
</AccordionGroup>
