## Using Play Mode

<AccordionGroup>
  <Accordion title="Entering Play Mode" defaultOpen={false}>
    1. Open an Editor file

    2. Select the **Play** icon in the top toolbar

    3. A new tab will automatically open, showing the scene in Play Mode
  </Accordion>

  <Accordion title="Seeing real-time edits in Play Mode" defaultOpen={false}>
    All edits made to an Editor file are instantly reflected in Play Mode. Set up a split-screen with Editor on one side and Play Mode on the other to see how edits impact a prototype in real-time.
  </Accordion>
</AccordionGroup>
