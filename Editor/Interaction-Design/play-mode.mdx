## Using Play Mode \[missing assets, bullets in 3 messed up]

<AccordionGroup>
  <Accordion title="Entering Play Mode" defaultOpen={false}>
    1. Open an Editor file

    2. Select the **Play** icon in the top toolbar

    3. A new tab will automatically open, showing the scene in Play Mode
  </Accordion>

  <Accordion title="Seeing real-time edits in Play Mode" defaultOpen={false}>
    All edits made to an Editor file are instantly reflected in Play Mode. Set up a split-screen with Editor on one side and Play Mode on the other to see how edits impact a prototype in real-time.
  </Accordion>

  <Accordion title="Sharing Play Mode prototypes" defaultOpen={false}>
    The Play Mode URL will automatically open the interactive prototype, not the Editor file. Any edits made to the Editor file will be reflected in the Play Mode prototype.&#x20;

    1. Follow the steps above to enter Play Mode

    2. Copy the Play Mode tab's URL&#x20;

    3. Send the Play Mode URL to the intended audience.&#x20;
  </Accordion>
</AccordionGroup>
