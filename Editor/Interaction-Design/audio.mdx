## Using Audio objects \[missing assets & numbering]

There are two ways to can add Audio object to a scene. Once added, Audio objects are identified in a scene by the purple speaker icon.

<AccordionGroup>
  <Accordion title="Adding Audio objects from the Objects menu" defaultOpen={false}>
    * Open the **Objects** menu in the top toolbar

    * Select **Audio** from the dropdown

    * Once selected, click in the scene to place the Audio object> An Audio track is automatically assigned to any Audio object added from the Objects menu. See instructions to change the Audio track below.&#x20;
  </Accordion>

  <Accordion title="Dropping an Audio file into the scene" defaultOpen={false}>
    1. Drag and drop an Audio file into the Editor scene

    2. The Audio track will automatically upload and be assigned to an Audio object

    > Audio objects dropped into a scene are **not** spatial by default. To make them spatial, go to the Audio section in the Properties Panel and check the 'Spatial' checkbox.
  </Accordion>

  <Accordion title="Changing an Audio track" defaultOpen={false}>
    1. Select the Audio object in the Hierarchy or in the scene

    2. Go to the Audio section in the Properties Panel

    3. Click on the name of the Audio track you want to change

    4. To upload a custom Audio file, select the **+** icon in the dropdown

    5. To use an Audio track from the Bezi Library, click the 'This File' dropdown and select 'Bezi Library'&#x20;
  </Accordion>

  <Accordion title="Supported Audio formats" defaultOpen={false}>
    The Editor currently supports MP3 and WAV formats.
  </Accordion>
