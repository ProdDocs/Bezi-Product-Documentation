## Using Audio objects \[missing assets & numbering]

There are two ways to can add Audio object to a scene. Once added, Audio objects are identified in a scene by the purple speaker icon.

<AccordionGroup>
  <Accordion title="Adding Audio objects from the Objects menu" defaultOpen={false}>
    Open the **Objects** menu in the top toolbar

    Select Audio from the dropdown

    Once selected, click in your scene to place the Audio object

    > An Audio track is automatically assigned to any Audio object added from the Objects menu. See instructions to change the Audio track below.&#x20;
  </Accordion>

  <Accordion title="Dropping a custom Audio file into the Editor scene" defaultOpen={false}>
    * Drag and drop an Audio file into the Editor scene

    * The Audio track will automatically upload and be assigned to an Audio object
  </Accordion>
</AccordionGroup>
