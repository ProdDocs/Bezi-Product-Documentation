## Adding States \[missing assets]

<Steps>
  <Step title="Open the State Machine panel">
    Select the **State Machine** button in the bottom bar to open the panel.

    \[asset]
  </Step>

  <Step title="Select the desired object">
    Select the object that will be made stateful. Do this by clicking on the object in the Hierarchy or directly in the scene.&#x20;

    <Tabs>
      <Tab title="Base State">
        The starting point when events should occur in a specific order. For example, if a sphere should grow every time you click on it.&#x20;
      </Tab>

      <Tab title="Any State">
        When an interaction should always produce the same result, independent of any interactions that came before. For example, if the same sphere should disappear every time `X` is pressed, regardless of its size.
      </Tab>
    </Tabs>

    Observe that every object always contains two state nodes:

    \[asset]
  </Step>

  <Step title="Add a new State to the object">
    Press the **+ New State** button in the State Machine panle to add a new state to the selected object
