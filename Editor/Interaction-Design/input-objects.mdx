*The Input object allows users to connect a gamepad, keyboard, or XR controller as a trigger input in the State Machine.*

*Inputs offer the ability to bind keys and buttons to actions. These actions can be used in the state machine to trigger events.*

## Using input objects \[missing assets]

Input objects can be bound to device keys or buttons to specific actions.

<AccordionGroup>
  <Accordion title="Supported input device types" defaultOpen={false}>
    Five devices are currently supported as input objects:

    1. Nintendo Switch

    2. Xbox

    3. PlayStation 5

    4. Keyboard and mouse

    5. Quest-based controllers
  </Accordion>

  <Accordion title="Adding input objects to a scene" defaultOpen={false}>
    Input objects can be added to a scene in two ways:

    1. Open the **Command Center**, type “input”, and select **Inputs** from the dropdown

    2. Open the **Objects** menu in the top toolbar and select **Inputs** from the dropdown
  </Accordion>
</AccordionGroup>
