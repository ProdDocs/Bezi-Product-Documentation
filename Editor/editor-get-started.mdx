## Command Center \[missing copy / asset]&#x20;

All actions can be accessed through the Editor's **Command Center**. There are three ways to activate it:

1. Start typing in the file

2. Type `/`

3. Click on the Bezi logo on the top left toolbar



Any previous action can be repeated by pressing `spacebar`

## Keyboard shortcuts

A full list of keyboard shortcuts can be found in your Editor file:

1. In your file, go to **Menu**

2. Select **Keyboard shortcuts** in the dropdown

## Navigation controls

How to navigate (orbit, pan, and dolly) your scene in the Editor

<AccordionGroup>
  <Accordion title="Mouse controls" defaultOpen={false}>
    Orbit: `right mouse button`

    Pan: `middle mouse button`

    Dolly (zoom in and out): `scrollwheel`
  </Accordion>

  <Accordion title="Trackpad controls" defaultOpen={false}>
    Orbit: `2 finger drag`

    Pan: `2 finger swipe`

    Dolly (zoom in and out): `2 finger pinch`
  </Accordion>

  <Accordion title="Right Mouse button controls" defaultOpen={false}>
    Orbit: `right mouse button`

    Pan: `space` + `right mouse button`

    Dolly (zoom in and out): `alt` + `right mouse button`

    Dolly to cursor: `control` or `command` + `right mouse button`
  </Accordion>

  <Accordion title="Autoswitch controls" defaultOpen={false}>
    Orbit: `right mouse button`

    Pan: `space` + `right mouse button`

    Dolly (zoom in and out): `shift` + `space` + `right mouse button`
  </Accordion>

  <Accordion title="Modifier behaviors" defaultOpen={false}>
    Unconstrain (Dolly to cursor): `cmd`

    Constrain (Dolly to center): `shift`
  </Accordion>
</AccordionGroup>
