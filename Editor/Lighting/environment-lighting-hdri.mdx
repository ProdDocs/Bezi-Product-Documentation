New files come with a default HDRI. To change the HDRI, follow these steps:&#x20;

<AccordionGroup>
  <Accordion title="Selecting a Bezi-provided HDRI" defaultOpen={false}>

    2. In the right sidebar, click the current HDRI name (e.g. Studio Interior)

    3. Click the 'This File' dropdown

    4. Click 'Bezi Library'

    5. Select the desired HDRI
