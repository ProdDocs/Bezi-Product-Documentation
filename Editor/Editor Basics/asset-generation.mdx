## Using Generate

<AccordionGroup>
  <Accordion title="Starting from Command Center" defaultOpen={false}>
    Type a prompt in the **Command Center** to activate the Generate panel. Once open, either select an existing asset from Community assets or generate a new custom asset.
  </Accordion>

  <Accordion title="Starting from Generate panel" defaultOpen={false}>
    Select the **Generate** tab in the bottom bar to open the panel. Once open, type a prompt into the search bar to find existing Community assets or press 'Generate' to generate new custom assets.&#x20;
  </Accordion>
</AccordionGroup>
