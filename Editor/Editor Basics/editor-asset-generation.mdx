## Using Generate

<AccordionGroup>
  <Accordion title="Starting in Command Center" defaultOpen={false}>
    Type an asset prompt in the **Command Center** to open the Generate panel. Once open, select an existing asset from the Community assets or generate a new one.
  </Accordion>

  <Accordion title="Starting in the Generate panel" defaultOpen={false}>
    Select 

    **Generate**

     from the bottom bar to open the Generate panel. Once open, type into the prompt field to search existing assets, or press 

    **Generate**

     to create a new asset.
  </Accordion>
</AccordionGroup>
