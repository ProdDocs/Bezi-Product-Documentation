## Accessing Modeling tools

To access Modeling tools you must be enter **edit mode**. This can be done in two ways:

1. Double-click on an object

2. Select an object and press `enter`

<Info>
  Parametric objects become ‘editable’ (bake its parameters) once you make edits with the Modeling tools.
</Info>

### Exiting edit mode

Exiting **edit mode** in the Modeling tool can be done in two ways:&#x20;

1. Double-click outside of the project

2. Press `enter`
