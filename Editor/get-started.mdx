All actions can be accessed through our **Command Center**. There are multiple ways to activate it:

* Start typing in the file

* Type `/`

* Click on the Bezi logo on the top left toolbar

To select/activate an action, you can press `spacebar`

### Repeating a previous action

* You can repeat a previous action by pressing `spacebar`
